# UV Layers Tools (ULT)

[![License: GPL v3](https://img.shields.io/badge/License-GPLv3-blue.svg)](https://www.gnu.org/licenses/gpl-3.0)

**ULT** — Blender аддон для массового управления UV-слоями.

## Возможности

- Добавление, удаление, переименование UV-слоёв на выделенных мешах.
- Перемещение UV-слоёв вверх/вниз на выделенных мешах (с сохранением данных развёртки).
- Информационная панель: отображает количество выделенных мешей и не-мешей, количество мешей без UV-слоёв, предупреждает о совпадении/несовпадении количества UV-слоёв и их имён между выделенными мешами. При определённых количествах выделенных объектов на информационной панели появятся забавные смайлики (¬‿¬)
- Выделение всех мешей в сцене и снятие выделения с не-мешей.
- Установка активного/рендерного UV-слоя по индексу.
- Синхронизация активного/рендерного UV-слоя между мешами.
- Автоматическая синхронизация рендерного UV-слоя с активным UV-слоем на выделенных мешах.
- Именование UV-слоёв с помощью пресетов имён (имеются встроенные пресеты, а также возможность создавать свои).
- Блок UV-слоёв активного меша с кнопками +/– (аналог нативного окна UV Maps).

## Совместимость

Аддон протестирован и корректно работает в этих версиях Blender: 3.6, 4.5, 5.1.

## Установка

1. Скачайте файл `UV_Layers_Tools_ULT.py` из [релизов](https://github.com/AbstracteurDeQuinteEssence/Blender-UV-Layers-Tools-ULT/releases).
2. В Blender: **Edit → Preferences → Add-ons → Install…**
3. Выберите скачанный файл и установите аддон.
4. Панель аддона находится в 3D Viewport → Sidebar (`N`) → вкладка **ULT**.

## Использование

- Выделите один или несколько мешей.
- Используйте информационную панель для получения сводки по выделенным объектам.
- Используйте кнопки на панели для массовых операций с UV-слоями.
- Применяйте встроенные пресеты имён для именования UV-слоёв или создавайте свои.
